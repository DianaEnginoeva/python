<h2>Django</h2>

Установка Django на macbook:

1. Verify that you have Python on your computer
```terminal
python --version
```
2. Verify that you have pip 
```terminal
pip --version
```
OR
2. Install pip
```terminal
curl -O https://bootstrap.pypa.io/get-pip.py
python get-pip.py
```
3. Using pip to install Django
```terminal
sudo pip install Django
```
